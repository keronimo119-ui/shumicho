## 詳しい説明(4000字以内)

```
■ 趣味帳は「あなたの好き」をしまう帳面アプリ

推し活の戦利品、集めているカード、出会った昆虫や花、食べ歩きの記録——
自分だけの図鑑を作って、番付でランキングして、ときには一句詠む。
3つの趣味ツールがひとつになった、毎日開きたくなる記録帳です。

データはすべて端末の中だけに保存。会員登録なし、すべての機能が無料で使えます。

──────────────────
📚 マイ図鑑 — なんでも自分だけの図鑑に
──────────────────
・写真とメモで「自分が出会ったもの」をコレクション図鑑化
・推しグッズ、トレカ、マンホールカード、昆虫、植物、食べ歩き…テーマは自由
・目標数を決めればコンプ率が進捗バーで見える
・「未入手(ほしいもの)」も登録できるので、収集リストとしても使える
・お気に入り度(★5段階)、入手日、場所、思い出メモを記録
・検索・並べ替えでさっと見返せる

──────────────────
🏆 なんでも番付 — 大相撲の番付表風ランキング
──────────────────
・好きなラーメン店、推し曲、温泉、なんでも番付に
・名前を入れて並べるだけで、横綱・大関・関脇…と自動で東西に振り分け
・本格的な縦書き番付表の画像を生成、SNSでシェアしたり保存したり
・「わが家の番付」を家族や友だちと見せ合うのも楽しい

──────────────────
🖌️ 俳句・川柳手帳 — 一日一句のすすめ
──────────────────
・「今日の季語」を毎日日替わりで表示(解説つき・約120語収録)
・五七五をさっと書き留めて、自分だけの句帳に
・縦書きの清書画像を生成。俳号(署名)と落款入りで本格的
・季節やお気に入りで絞り込み。川柳・自由律もOK

──────────────────
🔒 安心への配慮
──────────────────
・記録・写真はすべて端末内のみに保存(外部送信なし)
・会員登録・ログイン不要
・バックアップ用にデータの書き出し/読み込みに対応
・広告は画面下部のバナー1本だけ。操作のじゃまをしません

俳句は今日から、図鑑は思い出から、番付は今夜の晩酌のお供に。
あなたの「好き」を、趣味帳にしまってください。
```
