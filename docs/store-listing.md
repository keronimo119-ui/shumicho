## 詳しい説明(4000字以内)

```
■ 趣味帳Favは「あなたの好き」を毎日ひらく記録帳

推し活の戦利品、集めているカード、出会った景色、食べ歩きの記録——
自分だけの図鑑を作り、番付でランキングし、ときには一句詠む。
さらに写真を自由にデコって残し、カレンダーで毎日をふり返る。
いくつもの趣味ツールがひとつになった、毎日ひらきたくなる記録帳です。

データはすべて端末の中だけに保存。会員登録なし、すべての機能が無料で使えます。

──────────────────
📚 マイ図鑑 — なんでも自分だけの図鑑に
──────────────────
・写真とメモで「自分が出会ったもの」をコレクション図鑑化
・推しグッズ、トレカ、マンホールカード、昆虫、植物、食べ歩き…テーマは自由
・目標数を決めればコンプ率が進捗バーで見える
・「未入手(ほしいもの)」も登録できて収集リストにも
・お気に入り度(★5段階)、入手日、場所、思い出メモを記録

──────────────────
📖 軌跡 — 写真を自由にデコるスクラップ
──────────────────
・写真をステッカーのように貼って、拡大・回転・重ね順を自由自在
・角丸/丸/星の切り抜き、白フチの透過、手描きペン、2色カラー背景
・日記・名言・プロフィール・歌詞・想い・エピソードなどの書き込み枠
・年表づくりやプロフィールページづくりにもぴったり

──────────────────
👀 観覧モード — 作品をめくって楽しむ
──────────────────
・登録した宝物を1点ずつ、めくるように鑑賞
・写真は指で「ぷにぷに」触れる遊びごころのギミックつき
・2本指ピンチでズーム、作品はそのまま画像でシェアも

──────────────────
🏆 なんでも番付 — 大相撲の番付表風ランキング
──────────────────
・好きなラーメン店、推し曲、温泉、なんでも番付に
・名前を並べるだけで横綱・大関・関脇…と自動で東西に振り分け
・本格的な縦書き番付表の画像を生成、SNSでシェアや保存

──────────────────
🖌️ 俳句・川柳手帳 — 一日一句のすすめ
──────────────────
・「今日の季語」を毎日日替わりで表示(解説つき・約120語収録)
・毎朝の季語通知で、一句詠む習慣づくりをそっと後押し
・五七五をさっと書き留めて、写真を背景にした清書画像も作れる
・俳号(署名)と落款入りで本格的。川柳・自由律もOK

──────────────────
📅 カレンダー&手帳プレビュー — 毎日をふり返る
──────────────────
・図鑑・番付・俳句・軌跡の記録が日付に集まる連動カレンダー
・日付をタップすると、その日が1枚の「手帳ページ」に
・連続記録バッジで「続けた日々」が見える
・誕生日・記念日のイベント登録でカウントダウン

──────────────────
🔒 安心への配慮
──────────────────
・記録・写真はすべて端末内のみに保存(外部送信なし)
・会員登録・ログイン不要
・写真ごと丸ごとバックアップの書き出し/読み込みに対応
・広告は画面下部のバナー1本だけ。入力中は自動で隠れ、操作のじゃまをしません

集めて、めくって、デコって、一句。
あなたの「好き」の軌跡を、趣味帳Favに残してください。
```
