# データセーフティ回答メモ(Play Console)

AdMobバナーのみ使用・アプリ自体はデータ収集なし、という前提の回答。

## 概要の質問

| 質問 | 回答 |
|---|---|
| アプリはユーザーデータを収集または共有しますか? | **はい**(AdMob SDKが広告IDを収集するため) |
| 収集したデータはすべて転送中に暗号化されますか? | はい |
| ユーザーはデータの削除をリクエストできますか? | いいえ(アプリは端末外にデータを保持しないため。アンインストールで全削除) |

## 収集データの申告

### デバイスIDまたはその他のID(広告ID)
- 収集: **はい** / 共有: **はい**(Google AdMobと)
- 処理は一時的か: いいえ
- 必須かオプションか: 必須
- 目的: **広告またはマーケティング**

※アプリ独自に収集するデータは**なし**。写真・記録・俳句などの入力内容はすべて端末内保存で、収集にも共有にも該当しない(Playの定義では「端末外に送信されないデータ」は収集に当たらない)。

## 広告ID権限

- AndroidManifestに `com.google.android.gms.permission.AD_ID` がAdMob SDKにより自動付与される
- Play Consoleの「広告ID」質問には「はい、広告IDを使用します」+用途「広告」で回答

## コンテンツレーティング(IARC)

- 暴力・性的内容・薬物など: すべて「なし」
- ユーザー生成コンテンツの共有: 「なし」(画像生成はユーザーが自分で共有操作するのみで、アプリ内に投稿・交流機能はない)
- 広告の有無: 「あり」
